OpenAI 发布 GPT Agents：新一代 AI Agent 框架详解

OpenAI 正式发布 GPT Agents 框架，支持多步骤任务执行、工具调用和长期记忆保持。本文深入解析其核心功能和使用方法。

2026/03/15 5 分钟阅读
OpenAIAgentGPT

OpenAI 今日正式发布了 GPT Agents 框架，这是一个专为构建 AI Agent 应用而设计的全新框架。

核心特性

GPT Agents 框架提供了三大核心能力：

  • 多步骤任务执行：支持将复杂任务分解为多个步骤
  • 工具调用系统：内置工具调用接口，支持 API、代码执行
  • 长期记忆保持：创新的记忆系统，让 Agent 能够在多次对话中保持上下文

快速开始

from gpt_agents import Agent, tool

@tool
def search_web(query: str) -> str:
    return f"搜索结果: {query}"

agent = Agent(
    name="Research Assistant",
    tools=[search_web]
)

result = agent.run("帮我查找 2026 年 AI 的发展趋势")

适用场景

  1. 自动化研究和数据分析
  2. 智能助手和客服机器人
  3. 代码生成和调试辅助

GPT Agents 框架目前处于公开测试阶段，所有 API 在正式版发布前都可能发生变化。
